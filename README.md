Описание датасета
Датасет содержит отзывы действующих клиентов Т-Банка о качестве продуктов, услуг и клиентского сервиса. Данные собраны с внешних платформ (таких как Banki. ru, Sravni.ru) и социальных сетей (ВКонтакте, Одноклассники и др.), где клиенты публикуют свое мнение о взаимодействии с банком.
Каждый отзыв отражает реальный опыт клиента — от использования мобильного приложения до обращения в поддержку. Некоторые отзывы носят оценочный характер, другие описывают конкретные проблемы, требующие решения. Все такие обращения попадают в работу специалистам клиентской поддержки, которые взаимодействуют с клиентом и стараются решить его вопрос.
Датасет включает как количественные метрики (оценки, флаги, временные метки), так и качественные данные — полный текст отзыва, что позволяет проводить комплексный анализ: от сегментации до анализа текстов.


Общая информация
Период данных: Данные охватывают период с января по ноябрь 2025 года
Источники данных: Отзывы собраны с внешних платформ (Banki. ru, Sravni.ru) и социальных сетей (ВКонтакте, Одноклассники и др.), где клиенты Т-Банка публикуют свое мнение о продуктах и сервисах.
Процесс обработки: Все отзывы проходят модерацию. Обращения, содержащие жалобы или запросы на решение проблемы, направляются специалистам поддержки в социальных сетях. Сотрудники связываются с клиентом при необходимости, решают вопрос и фиксируют факт завершения.
Оценка клиентского опыта: Клиенты могут поставить оценку по шкале от 1 до 5.
На некоторых платформах оценка не указывается, но можно выбрать тип отзыва: позитивный, нейтральный или негативный.
На основе оценки и тональности формируется агрегированная метрика, близкая к NPS-подходу:
• Позитивные (промоутеры): оценки 4–5, а также нейтральные/позитивные без оценки;
• Негативные (детракторы): оценки 1–3, а также негативные без оценки.
CSAT (Customer Satisfaction Score): Уровень удовлетворенности клиента, рассчитанный на основе текста, оценки и контекста отзыва. Может использоваться для внутреннего мониторинга качества сервиса.


Текст отзыва: Поле review_text содержит оригинальный текст, оставленный клиентом. Это позволяет проводить текстовый анализ: выявление частых тем, автоматическую классификацию, анализ эмоциональной окраски и поиск скрытых проблем.

## Описание переменных

| Variable | Type | Description |
|----------|------|-------------|
| `review_dttm` | `datetime` | Дата и время начала отзыва (момент публикации или обращения) |
| `finish_dttm` | `datetime` | Дата и время завершения обработки отзыва (решения вопроса) |
| `id_client` | `integer` | Уникальный идентификатор клиента (если известен) |
| `company` | `categorical` | Направление, к которому относится отзыв: банк, страховая, инвестиции, работа и др. |
| `review_mark` | `integer` | Оценка, поставленная автором отзыва (если указана) |
| `review_emotion` | `categorical` | Эмоциональная окраска отзыва: позитивный, нейтральный, негативный |
| `review_source` | `categorical` | Платформа, с которой поступил отзыв: Banki.ru, ВКонтакте, Одноклассники и т.д. |
| `business_line` | `categorical` | Бизнес-линия, указанная в отзыве (например, кредит наличными, депозитные продукты) |
| `product` | `categorical` | Конкретный продукт, о котором идет речь (например, ипотека, дебетовая карта, вклад) |
| `reason` | `categorical` | Причина обращения (например, встречи, неверная консультация) |
| `review_theme` | `categorical` | Тематика отзыва (например, сервис, условия кредита, мобильное приложение) |
| `subtheme` | `categorical` | Подтема отзыва (более детализированная категория, например: «ошибка входа в приложение») |
| `review_text` | `text` | Полный текст отзыва клиента или комментатора |
| `solution_flg` | `integer` | Флаг: была ли решена проблема клиента (0/1) |
| `gender_cd` | `categorical` | Пол клиента (если автор — клиент и данные доступны) |
| `education_level_cd` | `categorical` | Уровень образования клиента |
| `marital_status_cd` | `categorical` | Семейное положение |
| `children_cnt` | `integer` | Количество детей у клиента |
| `citizenship_country` | `categorical` | Гражданство клиента |
| `segment_name` | `categorical` | Клиентский сегмент (например, Доходный, Защищенный) |
| `subscription_important_flg` | `integer` | Флаг наличия подписки «Премиум» или «Прайват» (1 — есть) |
| `new_flg` | `integer` | Флаг нового клиента: первый продукт открыт не более 3 месяцев назад (1 — новый) |
| `influencer_flg` | `integer` | Флаг VIP клиента |
| `age_segment` | `categorical` | Возрастная группа клиента (например, молодежь, взрослые, сеньоры) |
| `csat_score` | `float` | Уровень удовлетворенности клиента (Customer Satisfaction Score), рассчитанный на основе отзыва |
